Googlefy
========

Plugin para Firefox que remarca resultado conforme a dominio seleccionado.

Instalación
-----------

+ En el menú archivo seleccione abrir archivo (Crl-O), navegue hasta la carpeta build y seleccione googlefy-1.1-an+fx.xpi.
+ Se le solicitará permiso para instalar el complemento, click en Añadir.


Uso
---

+ En una pestaña cualquiera de su navegador Firefox acceda a google.com.
+ Realice una búsqueda.
+ Click en el icono de Googlefy. 
+ Inserte una url en el campo vacío y de click en enviar.
